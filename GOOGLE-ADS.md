## Campaign Settings

| Setting | Value |
|---|---|
| Budget | $20-30/day shared |
| Bidding | Manual CPC, start at $1.50-2.00 |
| Network | Search only |
| Location | United States |
| Match types | Exact + Phrase only |

## Negative Keywords

```
free
```

- `[State] Lease Agreement — $5`
- `Download Your PDF in 60 Seconds`
- `No Account Needed`
- `State-Specific & Legally Compliant`
- `Instant Residential Lease Agreement`
